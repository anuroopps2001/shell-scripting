
# Multiway statements (CASE statements)
- Syntax
```bash
case value in 
label 1) statements ;;
label 2) statements ;;
.
.
.
.
*) statements ;; --> (default case)
esac
```
Here, Value can be any word, expression or integer
and label can be any word, integer or an expression

;; is used for terminating the control from case block
